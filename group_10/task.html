1 Что такое html-элемент?

<html> - элемент это основная конструкция в языке html.
Состоит из 3х составляющих: открывающий тег, содержимое, закрывающий тег. Содержимым может быть текст или другой элемент. В открывающем теге содержатся атрибуты тега. Есть элементы, которые не имеют содержимого и закрывающего тега.

2 Перечислите основные теги. Что они делают?

  • <html> - содержит всю страницу
  • <head> - содержит служебную информацию для браузера
  • <body> - основе содержимое, видимое пользователю
  • <p> - абзац
  • <h1> - <h6> - заголовки
  • <a> - гиперссылка
  • <img> - вставка картинки
  • <style> - вставка стилей на страницу

3 Что такое атрибут тега?

Атрибут тега пишется после имени тега в открывающем теге элемента.
Атрибутов может быть сколько угодно в открывающем теге. Они разделяются пробелами.
Атрибут пишется так: href="./my_page.html"

4 Какую структуру имеет веб-страничка(html страничка)?

html-страница имеет структуру дерева.
Есть главный элемент- корневой элемент, а все остальные вложены в него и друг в друга.

5 Что такое css?

css - каскадные таблицы стилей.
Нужны нам, чтобы добавить красоту и управлять внешним видом наших элементов.

6 Как использовать css?(где писать)

  • внутри тега style
  • отдельным файлом
  • атрибутом style html - тега
        селектор {
            имя-свойства: значение-свойства;
        }

7 Что такое css селектор. Какие селекторы вы знаете?

Селектор позволяет выбрать элементы html языком css, чтобы потом применить к ним css-свойства.
Простые селекторы:
по имени тега
по id
по классу
